# Sistema de Eventos Programação
Sistema Web criado para simular um site de eventos educacionais sobre programação
<br>

---

# **Ferramentas Utilizadas**
## Ferramentas Front:
- HTML e CSS:
    - Ferramenta de marcação mais comum para desenvolvimento front-end.

- JavaScript:
    - Ferramenta utilizada para criação da lógica do front-end

- Mais informações detalhadas sobre o front-end e o porque das escolhas de ferramentas:
[Front-End](docs/front.md)

## Ferramentas Back:
- Express.js:
    - Ferramenta utilizada para criação do back-end, criando o servidor com as rotas e a conexão com o banco de dados.
- Mais informações detalhadas sobre o back-end e o porque das escolhas de ferramentas: 
[Back-End](docs/API.md)

## Ferramentas Bancos de Dados:
- MySql:
    - Ferramenta utilizada para criação de banco de dados relacionado.
- MongoDB:
    - Ferramenta utilizada para criação de banco de dados não relacionados.

- Mais informações detalhadas sobre os bancos e o por que das escolhas de ferramentas:
[Bancos de Dados](/docs/bancos.md)

## Ferramentas de teste:
- Postman:
    - Ferramenta utilizada para fazer testes de API.
- Mais informações detalhadas sobre os testes e o por que das escolhas de ferramentas:
[Testes](docs/testes.md)
